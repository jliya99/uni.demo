Computer Science & Engineering (CSE)

About CSE

Computer Science and Engineering (CSE) is one of the most popular departments in modern universities. It focuses on programming, software development, artificial intelligence, web development, database management, networking and cybersecurity.

Students of CSE learn different programming languages such as C, C++, Java, JavaScript and Python. They also develop problem solving skills and create software applications for real-world use.

Career Opportunities

  • Software Engineer
  • Web Developer
  • App Developer
  • Database Administrator
  • Network Engineer
  • Cyber Security Specialist
  • AI Engineer
